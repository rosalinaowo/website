Setup

Server

This website and all the other services I host run on my HP Omen Laptop. The OS of my choice is Debian for it's stability and reliability.

All of my services use HTTPS and I keep no logs so no one will be spying on you.

If you don't want to connect over the clearnet you can use my TOR hidden service.

Also shoutout to Free DNS, since I use them as my Dynamic DNS because I don't have a static IP.

Personal computer

I use a pre-built desktop with some modifications. (Specs: i7-9700, 16GB 2666MHz, RTX 2070, 2.5TB of total storage, MasterLiquid 2X). I daily drive Arch Linux, but I have Windows installed on another disk and use it only for gaming.

As for the software, my browser is Firefox (even though I should use LibreWolf) with a custom profile to disable spyware and other "features", my editors are Neovim and Lite XL and I virtualize with KVM.
